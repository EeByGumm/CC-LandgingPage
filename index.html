Raspberry Pi Foundation

To put the power of computing and digital making into the hands of people all over the world.

Single Board Computing

Small form factor computing on a board about the size of a credit card

Powered by ARM architecture

An ARM processor utilises reduced instruction set computing (RISC) design, this is paired with a simplified central processing unit (CPU) which gives high performance.

Operated by Raspberry O/S

The native operating system is Raspberry Pi OS a linux operating system which can boot to command line or a GUI desktop.
